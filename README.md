# Notes 学习笔记

#### 介绍

- 这是一款学习笔记软件，用于记录在编码过程中学习到的知识，方便复习。在浏览项目的过程中还会有练习题目弹出，进行==巩固==学习。

#### 技术栈

- python flask为后端

- html, css, js为前端
